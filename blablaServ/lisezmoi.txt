Pour démarrer le serveur (port 18000 par défaut) :

Windows : double-cliquez sur startBlabla.bat
Unix/Linux : en ligne de commande, placez vous dans ce repertoire et tappez : "sh startBlabla.sh" ou "./startBlabla.sh" 

La commande suivante fonctionne partout : java -jar BlablaServ.jar [numport]

---

Pour lancez le client de chat (il ne fonctionne qu'en local sinon modifiez l'ip dans le fla) :

Windows : double-cliquez sur blablachat.swf
Unix/Linux : ouvrez le swf avec votre navigateur web
